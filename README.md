# MPIC Website & Student Portal

A modern responsive website and student management platform for Mahadev Prashad Inter College (MPIC), Chauri Chaura.

## Features

### Public Website

- Modern responsive homepage
- About section
- Facilities showcase
- Gallery
- Contact form

### Student Portal (Planned)

- Student login
- Online results
- Attendance tracking

## Tech Stack

### Frontend

- React
- Vite
- Tailwind CSS

### Backend (Planned)

- Firebase
- Spring Boot
- MySQL
